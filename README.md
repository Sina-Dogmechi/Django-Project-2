## A small online shop project that is more efficient than the social network project ([Django-Project-1](github.com/Sina-Dogmechi/Django-Project-1)) and more techniques are used in it.
> [!NOTE]
> _All the packages used in this project are located in the <ins>requirements.txt</ins> file for installation in the virtual environment._

**In addition to the items mentioned in the previous project (Django-Project-1), there are also the following new items in this project:**
- Set up a custom user model
- Registration with verification code using the Kavenegar SMS service
- Using django-storages for media files with Abrarvan service
- Limiting access to logged-in users that pass a test (user_passes_test)
- Celery beat + Daemonization
- Context processors
- Permissions (define and apply to users)
- Replacing the Postgresql with the default database (sqlite)
- Creating a cache using Redis and connecting session related issues to it
